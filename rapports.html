✮ MON RAPPORT ✮




( ✮ WHAT I LEARNED )

Grâce à ce projet, j'ai appris les bases de deux languages informatiques. Tout d'abord, HTML (Hypertext Markup Language) qui a pour but de créer des sites web. J'ai appris beaucoup de termes et leur utilisation tel que "div", "header, "main", "body", "footer", "p", "a", "br", "ul", "li" etc... J'ai aussi appris le CSS (Cascading Style Sheets) qui lui, est plus focalisé sur l'organisation det l'esthétique du site sur lequel on travaille. Evidemment, pour créer un site beau à voir, il faut passer par HTML et CSS donc on peut dire qu'ils sont liés, en quelque sorte.



( THOUGHTS ✮ )

J'ai beaucoup aimé faire ce projet. J'en ai moi-même été surprise étant donné que c'était du travail scolaire à faire pendant les vacances. Malgré que ce que je compte faire plus tard n'a pas forcément de lien avec l'informatique et la programmation, je pense que HTML et CSS pouront m'être utiles un de ces jours. C'était aussi très amusant de voir par quelles étapes passent les personnes qui créent les sites que nous critiquons si souvent.
